## Download

Clicca per accedere al link di download

- [Eclipse Enterprise](https://www.eclipse.org/downloads/packages/release/2024-06/m1/eclipse-ide-enterprise-java-and-web-developers) - IDE
- [Github Desktop](https://desktop.github.com/) - Gestione dei repository Git
- [JDK 17 Temurin](https://adoptium.net/temurin/releases/?version=17) - Java Development Kit 17
- [Maven](https://maven.apache.org/download.cgi) - Strumento di build automation
- [Project Lombok](https://projectlombok.org/download) - Utilizzo delle Annotation
- [Rancher Desktop](https://rancherdesktop.io/) o [Podman](https://podman.io/) - Gestione dei container Docker
- [Open VPN](https://openvpn.net/community-downloads/) - Configurazione VPN

### Eclipse Spring Boot
- Navigare nella sezione "Help" dell'IDE e poi "Eclipse Marketplace..."
- Installare "Spring Tools 4"

### Installazione Lombok su Eclipse
- Eseguire l'installer scaricato precedentemente di "Project Lombok"
- Cliccare su "Specify location..."
- Selezionare l'eseguibile "eclipse.exe" (l'ide di riferimento)
- Cliccare su "Install"
